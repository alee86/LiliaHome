# Lilia HOME - E-commerce de Aromatizantes

E-commerce moderno para la venta de aromatizantes premium con panel de administración integrado.

## 🌸 Características

- **Tienda Online** - Catálogo de productos con carrito de compras
- **Integración WhatsApp** - Confirmación de pedidos directamente por WhatsApp
- **Panel Admin** - Gestión de productos, carrusel e imágenes
- **Diseño Responsivo** - Optimizado para móvil y escritorio
- **Animaciones Suaves** - Transiciones elegantes y experiencia fluida

## 🚀 Inicio Rápido

### Requisitos
- Node.js 16+
- npm o yarn

### Instalación

```bash
npm install
```

### Desarrollo

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

### Build para Producción

```bash
npm run build
```

## 📁 Estructura del Proyecto

```
src/
├── App.jsx          # Componente principal (tienda y admin)
├── main.jsx         # Punto de entrada
index.html           # HTML base
package.json         # Dependencias
```

## 🎨 Design System

- **Paleta de Colores**: Tonos cálidos y naturales (cream, sand, taupe, brown, dark)
- **Tipografías**: Dancing Script (títulos), Cinzel (encabezados), Lato (cuerpo)
- **Componentes**: Botones, cards, modales, drawer del carrito

Ver `lilia-home-design-system.html` para detalles completos.

## 🔐 Acceso Admin

- URL: Haz clic en "Admin" en la tienda
- Contraseña: `admin123`
